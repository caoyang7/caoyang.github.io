# 一、Markdown概念 

Markdown 文档本质上是一种纯文本文档，所以没办法直接在文档里设置不同的文字大小、颜色、位置，也不能直接显示图片、表格等，但可以通过一些简单易记的符号用来标识这些格式和元素，然后通过Markdown渲染引擎就可以生成图文并茂、格式丰富的文档。
